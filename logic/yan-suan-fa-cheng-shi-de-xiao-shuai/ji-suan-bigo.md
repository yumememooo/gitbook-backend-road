# 計算BigO

### 計算BigO

{% hint style="info" %}
_BigO (_**大O符號**（Big O notation) 又稱漸進符號

電腦科學中_，_ 用來分析程式碼演算法的時間複雜度非常有用
{% endhint %}

O(n)不代表`運行時間`，它代表的是`執行時間隨著數據規模增長的變化趨勢`。



* O(1)
* O(logn)&#x20;
* O(n) 一個迴圈
* O(nlogn)&#x20;
* O(n^2)
* O(n!)
* O(2n)
