# git

### 有關vscode插件

Git Graph [https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph](https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph)

這個插件可以快速讓你切換管理分支，可以查看圖表，設定git，merge，Tag等等操作．



#### git commit
